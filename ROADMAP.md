# Farsi - Post-MVP Roadmap

Future improvements for the Farsi ecosystem after the PL Genesis Hackathon:

1. **Real Yield Integration**: Move from simulated vaults to live integrations with lending protocols (e.g., Aave or Mento/Celo-native yield routes).
2. **Smart Accounts (ERC-4337)**: Implement full account abstraction to allow email/social login for even lower friction.
3. **Push Notifications**: Use Web Push API to notify users when they earn yield or when a friend contributes to a Social Pot.
4. **Native Mobile Port**: Use the existing React components to launch native iOS/Android apps via Capacitor or React Native.
5. **Fiat Off-Ramp**: Complete the circle by allowing users to sell USDC back to their bank accounts.
6. **Governance**: Introduce a community pot for sponsoring local Farsi-user projects.
